# Botify

AI-конструктор приложений на базе bolt.diy, адаптированный под Telegram Mini App с монетизацией через кредиты.

---

## Бизнес-модель (ВАЖНО — запомнить навсегда)

### Ключевое правило
- **Пользователи НЕ вставляют свои API ключи**
- AI работает только через **ключи владельца** (дешёвые модели: Gemini Flash, DeepSeek, Groq)
- Пользователи **покупают кредиты** и тратят их на генерации

### Себестоимость
- 1 генерация ≈ 4000 токенов → **~0.15–0.20 ₽/генерация** (Gemini Flash / DeepSeek)
- 1 кредит = 1 запрос к AI (создание/правка кода)
- Просмотр, чат без генерации = бесплатно
- Ошибка AI → кредит не списывается

## Технический стек

- **База**: bolt.diy (open-source AI web dev agent)
- **Фронтенд**: Remix + React + Vite
- **Стиль**: Tailwind CSS + UnoCSS
- **Порт**: 5000
- **Запуск**: `NODE_OPTIONS='--require ./polyfill-node18.cjs' CHOKIDAR_USEPOLLING=1 CHOKIDAR_INTERVAL=3000 pnpm run dev`

## Пользовательские предпочтения

- Язык интерфейса: русский
- Бренд: Botify (⚡ Botify)
- Пользователи не могут вставлять свои AI ключи — только владелец
- Монетизация через кредиты, не через подписку на инструменты
